========
Glossary
========

.. glossary::

    Glyphmath
        RoboFab objects support basic mathematical operations. Glyphs can be added, substracted and multiplied. The operations apply to all coordinates and all points. See also :doc:`how to use GlyphMath <../docs_howtos/glyphmath>`.

    NoneLab
        Name for RoboFab and python environment which is independent of FontLab. If something works in NoneLab it means that it can run on any platform that runs python with RoboFab installed.
